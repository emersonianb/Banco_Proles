Sobre o Proles

O Proles Bank foi um banco desenvolvido para a disciplina de POO - Programação Orientada a Objetos, no curso de Engenharia da Computação, no Instituto Federal de Campina Grande. O projeto teve seu back-end feito em C++, que era a linguagem utilizada na cadeira. Então, aproveitando o desenvolvimento desse projeto, decidiu-se desenvolver uma página web para o banco, para ajudar no aprendizado e, também, para passar o tempo.
